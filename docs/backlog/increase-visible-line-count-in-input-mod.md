Feedback input mode currently renders too few lines, making it hard to review or edit longer feedback before submission. Exact line count / sizing policy TBD — pick a sensible default (e.g. grow with content up to a cap, or a larger fixed height) and confirm it doesn't crowd out the surrounding dashboard panels. Input mode lives in the curses dashboard feedback flow; ground changes there rather than inventing new UI.
